Password Security Guidelines

  • Minimum Length: Use a minimum password length of 8 or more characters if permitted.
  • Character Variety: Include lowercase and uppercase alphabetic characters, numbers and symbols if permitted.
  • Random Generation: Generate passwords randomly where feasible rather than using predictable patterns.
  • Unique Passwords: Avoid using the same password twice across multiple user accounts and/or software systems.
  • Avoid Patterns: Avoid character repetition, keyboard patterns, dictionary words, letter or number sequences.
  • Personal Information: Avoid usernames, relative or pet names, romantic links (current or past), and biographical information.
  • Guessable Info: Avoid using information that colleagues or acquaintances might know to be associated with you.
  • Simple Combinations: Do not use passwords which consist wholly of any simple combination of the aforementioned weak components.
